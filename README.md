# 🍔 Food Delivery Time Prediction (Machine Learning + Flask)

## 📌 Project Overview
This project is an end-to-end Machine Learning application that predicts food delivery time based on multiple real-world factors such as delivery partner details, distance, order type, and vehicle type. The system is deployed using Flask and provides real-time predictions through a user-friendly web interface.

---

## 🚀 Features
- Predicts delivery time using Machine Learning models  
- Supports real-time user input through a Flask web app  
- Includes user authentication (Login/Signup system)  
- Interactive dashboard for analytics and insights  
- Handles categorical data using encoding techniques  

---

## 🧠 Machine Learning Workflow
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering & Encoding  
4. Model Training & Evaluation  
5. Model Deployment using Flask  

---

## ⚙️ Algorithms Used
- Linear Regression

- LightGBM
